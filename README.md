# Sobre o Projeto
O Projeto Cadastro é uma aplicação web para a inclusão de um cadastro de cliente.

# Tecnologias Utilizadas
Backend: .NET Core 6.0 MVC
Frontend: Bootstrap

# Instruções
Ao rodar a aplicação, irá carregar a tela de lista de Clientes. Para cada cliente, teremos dois botões('Editar' e 'Excluir'), 
onde será possível excluir ou editar suas informações.

Teremos também o botão "Novo Cliente", onde será possível incluir um novo cliente com as informações de Nome Completo, Email, CPF e Data de Nascimento. 
Todos os campos tem as validações que foram solicitadas no teste.
